"I Need to Think About It"

  1. Empathize: "Totally fair."
  2. Test belief: "Do you feel this would actually help you get to [goal]?"
  3. Isolate: "What specifically do you need to think about?"
  4. Handle: Address the real concern.
  5. Compress timeline: "When would you have an answer — tomorrow morning?"
  6. Set accountability: "Cool, I'll call you at 10am."
